Configure Expressium

Configuration

The Configuration provides an overview of the project properties used by the Expressium application. In the Enroller Actions section, an additional login sequence can be defined to simplify the initialization of the browser during the enrollment process. Custom controls may be specified for handling types of non-standard web elements and may be assign to page controls in the project editor. These custom controls will though require an appropriate implementation within the framework after code generation.
