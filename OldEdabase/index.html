Enhance the dynamic world of a PCB design

EDAbase is an Electronic Design Automation software application aimed for Printed Circuit Board designs. This unique product embrace advanced viewer and editor functionality and directly interface to both Mentor Graphics Expedition and DDE-EDA Supermax ECAD environments.

  • EDAbase - Is interfacing directly to a DDE - EDA A/S Supermax ECAD environment...
  • EDAbase - Is an enhanced PCB viewer including unique extra functionality...
  • EDAbase - Is a semi-automatically PCB placer providing full pre-placing ability...
  • EDAbase - Is a true Windows application with an easy-of-use graphical user-interface...
  • EDAbase - Is a single stand alone program easy to install, administrate and distribute...
  • EDAbase - Is a supporting both old and new Supermax ECAD design file formats...
