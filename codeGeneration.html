Code Generator

The Code Generator creates page objects and UI test classes within the IDE solution based on the content of the Expressium object repository. These UI tests are designed as non-destructive smoke tests for the web application and includes reusable code snippets for the subsequent development of the business test. Although Expressium generates most of the framework source code, an initial verification and minor customization of the page objects and UI tests may be required after code generation.
