Download

To utilize an Expressium created test automation framework, you need to install one of the supported IDEs - Visual Studio or Visual Studio Code. The community editions of these IDEs are fully adequate for this purpose. Download and install your preferred IDE along with the Expressium application before starting your journey. The application must be installed in a directory with both read write access as Selenium and Playwright drivers will be cashed locally during enrollment. The Expressium application can be uninstalled at any time by executing the uninstall.exe file located in the installation directory.
